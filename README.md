# WhiteboardFox Auto Draw Ultra Turbo

WhiteboardFox Auto Draw Ultra Turbo is a browser userscript for https://r2.whiteboardfox.com/ that automatically renders solid black-and-white images onto the whiteboard canvas.

The script is optimized specifically for high-contrast monochrome drawings and does not support grayscale or color images.

## Features
- Automated drawing directly onto the WhiteboardFox canvas
- Solid black-and-white rendering only
- Designed for clean, high-contrast output
- Lightweight and browser-based
